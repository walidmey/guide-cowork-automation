# Partie 3 — Automatiser avec les tâches planifiées

Les tâches ponctuelles, c'est bien. Les tâches qui tournent sans que vous y pensiez, c'est mieux.

Le système de tâches planifiées de Cowork permet de définir une tâche une seule fois et de la faire exécuter automatiquement sur un calendrier : tous les jours à 8h, chaque lundi matin, le premier du mois, toutes les heures entre 9h et 18h.

Cette partie couvre deux aspects :

1. **Comment créer et configurer une tâche planifiée** — l'interface, les options, les contraintes à connaître
2. **10 tâches prêtes à l'emploi** — des instructions complètes que vous pouvez copier et adapter immédiatement

La combinaison des deux vous permettra de déléguer votre premier workflow automatique dans l'heure.

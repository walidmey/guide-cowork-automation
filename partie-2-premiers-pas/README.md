# Partie 2 — Premiers pas

La configuration initiale de Cowork prend environ 30 minutes. Elle est déterminante : un projet bien configuré réduit de moitié le travail nécessaire pour chaque tâche future.

Cette partie couvre trois étapes :

1. **Installation et accès** — télécharger, connecter, choisir votre plan
2. **Créer un projet et définir le contexte** — le CLAUDE.md, la sélection du dossier, les instructions permanentes
3. **Votre première tâche déléguée** — une tâche concrète que vous pouvez tester dans l'heure

Ne sautez pas la configuration du projet. C'est là que se joue l'efficacité à long terme.

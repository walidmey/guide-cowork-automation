# Installation et configuration initiale

{% hint style="info" %}
**Durée estimée :** 30 minutes

**Cette partie couvre :**

* l'installation et l'accès à Cowork
* la création d'un projet et la définition du contexte
* votre première tâche déléguée
{% endhint %}

La configuration initiale de Cowork est déterminante : un projet bien configuré réduit de moitié le travail nécessaire pour chaque tâche future.

{% hint style="warning" %}
Ne sautez pas la configuration du projet. C'est là que se joue l'efficacité à long terme.
{% endhint %}

***

## Avant de commencer

* [ ] Vous avez accès à votre compte Cowork
* [ ] Vous avez choisi un dossier de travail dédié sur votre ordinateur
* [ ] Vous savez quel premier projet vous allez configurer
* [ ] Vous avez 30 minutes sans interruption pour faire la configuration

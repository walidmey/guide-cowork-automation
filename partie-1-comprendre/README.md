# Vue d'ensemble : ce que Cowork peut faire

{% hint style="info" %}
Cette page répond à une question clé : pourquoi Cowork et pas simplement Claude chat ? En 5 minutes, vous comprendrez la différence — et vous saurez quand utiliser l'un ou l'autre.
{% endhint %}

Avant de configurer quoi que ce soit, il faut comprendre ce qu'est vraiment Cowork — et ce qu'il n'est pas.

Cette partie répond à deux questions fondamentales :

1. **Qu'est-ce que Cowork fait que Claude chat ne peut pas faire ?** Les différences ne sont pas cosmétiques — elles changent ce qu'on peut lui déléguer.
2. **Comment Cowork agit-il concrètement ?** Quatre modes d'action distincts, que vous choisirez selon la tâche.

{% hint style="success" %}
Comprendre ces fondamentaux évite les erreurs classiques des nouveaux utilisateurs : essayer de faire faire à Cowork ce qu'il fait mal, et passer à côté de ce qu'il fait exceptionnellement bien.
{% endhint %}
